# Energy-UA ClockDesk Plugin ⚡⏳

**Неофіційний** плагін для додатку **ClockDesk**, який відображає графік відключень електроенергії, парсячи дані з сайту `energy-ua.info`.

> ⚠️ **Відмова від відповідальності (Disclaimer):**
> Цей додаток є повністю неофіційним проєктом з відкритим вихідним кодом. Розробник цього плагіну **не має жодного відношення** до ресурсу `energy-ua.info`, обленерго або будь-яких державних установ.
> Плагін лише автоматизує перегляд загальнодоступної інформації з веб-сайту. Використовуйте на власний ризик.

## 🚀 Можливості

* **Інтеграція з ClockDesk:** Працює як "Chip" (віджет) у додатку ClockDesk.
* **Зворотний відлік:** Показує точний час, що залишився до увімкнення або вимкнення світла (наприклад, "Є | до 14:30 | 45:00 хв.").
* **Офлайн режим:** Автоматично кешує графік на 20 хвилин. Якщо зникне інтернет, плагін покаже збережені дані.
* **Налаштування черги:** Ви можете вказати посилання на будь-яку чергу/групу через налаштування плагіну.
* **Перегляд деталей:** При натисканні на віджет відкривається вікно з повним графіком на добу.

## ⚙️ Налаштування

1.  Встановіть плагін та увімкніть його через налаштування **ClockDesk**.
2.  Натисніть на віджет, щоб його налаштувати
3.  Знайдіть свою чергу на сайті [energy-ua.info](https://energy-ua.info/).
4.  Скопіюйте посилання на вашу групу (наприклад: `https://energy-ua.info/cherga/5-2`).
5.  Вставте посилання у поле вводу в додатку та натисніть **"Зберегти та Перевірити"**.
6.  Якщо графік завантажився успішно, він автоматично з'явиться у ClockDesk.

## 🛠 Технічні деталі

Проєкт написаний на **Kotlin** і використовує наступні технології:

* **Jsoup:** Для парсингу HTML-структури сайту.
* **WebView & Coroutines:** Для асинхронного завантаження сторінок та виконання JavaScript (необхідно для сайтів із захистом від ботів).
* **Gson:** Для серіалізації та кешування даних у JSON.
* **BroadcastReceiver:** Для обміну даними з основним додатком ClockDesk.
* **Material Design:** Для інтерфейсу налаштувань.

## 📦 Як зібрати проєкт

1.  Клонуйте репозиторій.
2.  Відкрийте проєкт в **Android Studio**.
3.  Дочекайтеся синхронізації Gradle.
4.  Запустіть `Build` -> `Make Project` або `Run` на вашому пристрої.
